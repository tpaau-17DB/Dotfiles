# Dotfiles

**My Hyprland desktop dotfiles, feel free to use in your own rices!**


## My Setup

* OS: Arch Linux
* Shell: Fish
* Terminal: Kitty
* DE: Hyprland
* Bar: Waybar
* Font: Hack Nerd Font
* Launcher: Wofi
* Lock: Swaylock
* Session Management: Wlogout
* Notifications: Mako
* IDE: Neovim


## Project Status

This project is still in development. It may contain bugs and unfinished features.
